
现在有一个数列N，分别为A1,A2,A3.......AN，现在取一个三元组(i,j,k) （1 <= i < j < k < N）

现在要求统计满足以下条件的三角形

Ai,Aj,Ak各不相同，并且三角形的三边为Ai,Aj,Ak。

1≤N≤100

1≤Ai≤1e9

1≤Aj≤1e9

1≤Ak≤1e9

Input Format

5

4 4 9 7 5

Constraints

输入： 第一行一个数 N 第二行N个数：A1,A2,A3.....AN

输出： 选出满足条件的三角形的个数

Output Format

5

Sample Input 0

5
4 4 9 7 5
Sample Output 0

5
